OpenC1 v1.4
====================================================
BLOG: blog.1amstudios.com
CODE: code.google.com/p/openc1
EMAIL: jeff@1amstudios.com


OpenC1 is a ground-up remake of the classic driving and killing game by Stainless Software.


Prerequisites:
 1. Microsoft C++ runtime - http://www.microsoft.com/downloads/en/details.aspx?familyid=A5C84275-3B97-4AB7-A40D-3802B2AF5FC2
 2. XNA 3.0 - http://www.microsoft.com/downloads/en/details.aspx?FamilyID=6521d889-5414-49b8-ab32-e3fff05a4c50
 3. Nvidia PhysX - http://www.nvidia.com/object/physx-9.10.0513-driver.html
 4. Read and comply with license.txt


Keys:
 * Up, Down, Left, Right - Accelerate, brake, steer
 * Space - Handbrake
 * Backspace - Repair
 * C - Change camera
 * R - Reset vehicle
 * F4 - Edit modes (for debugging)
 * P - Take screenshot
